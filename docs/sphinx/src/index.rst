Penrose-Lamarck
================

Penrose-Lamarck is a small learning application built for Roche engineering
hiring. It is not a production product; it is a working codebase that
candidates extend as part of the evaluation. The system exposes
exercise-ingestion, practice-session, and metrics workflows through REST and
MCP interfaces, with Postgres storage, a web UI, and lightweight orchestration
and observability. The full API surface is intended to be accessed through an authorized MCP client.
