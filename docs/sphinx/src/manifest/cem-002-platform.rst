Platform and Service Recommendations (CEM-002)
==============================================

This section provides prescriptive guidance on the use of specific technologies
and services for common engineering needs.

Relational Data Storage (CEM-002-000)  
---------------------------------------------------

Specification
~~~~~~~~~~~~~

- Use **Postgres** in AWS for relational datastores.

Motivation
~~~~~~~~~~

- Modern SQL databases have equivalent performance for JSON data types.
- Postgres offers robust JSON support across relational databases.
