Coding Agents (CEM-008)
=======================

Codex (CEM-008-000)  
------------------------------

Specification
~~~~~~~~~~~~~

The OpenAI Codex is a recommended coding assistant. You can login in VS Code
with SSO via Roche credentials. Download the extension.

.. note::
   Codex requires a plus subscription. Access process is a WIP; follow internal threads for updates.

Motivation
~~~~~~~~~~

- Augmented coding assistant integrated with VS Code.

Claude code / ONA (CEM-008-002)  
------------------------------------------

Specification
~~~~~~~~~~~~~

Claude code is a terminal-based coding assistant. Access via ONA using SSO via
Roche credentials. Once a repository is accessed you can open Claude code via a
terminal using the ``claude`` command.

Motivation
~~~~~~~~~~

- Augmented coding assistant integrated with a secure development environment accessible through JetBrains, VS Code, Cursor, or SSH.
